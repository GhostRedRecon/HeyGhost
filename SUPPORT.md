# Support

For setup help, bugs, and feature requests, use GitHub Issues:

https://github.com/GhostRedRecon/HeyGhost/issues

Before opening an issue, include:

- Linux distribution and version.
- Hardware details.
- Python version.
- Ollama model used.
- STT/TTS backend details.
- Relevant logs or test output.
