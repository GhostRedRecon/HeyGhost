# Code Of Conduct

HeyGhost aims to be a respectful, educational, and safety-conscious project.

## Expected Behavior

- Be respectful and constructive.
- Focus on improving the project.
- Respect user safety, privacy, and consent.
- Keep cybersecurity discussion defensive and authorized.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Instructions or contributions intended for unauthorized access, malware, credential theft, or abuse.
- Publishing private information, secrets, or exploit details for unresolved vulnerabilities.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block users who violate this code of conduct.
